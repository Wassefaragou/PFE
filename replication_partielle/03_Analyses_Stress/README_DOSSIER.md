# Dossier 03 - Analyses de stress

Ce dossier contient les scripts dedies a la lecture du comportement du modele pendant les phases de stress du marche.

## Fichiers

- `analyze_stress_24_25.py` : analyse ciblee sur 2024-2025.
- `analyze_stress_rolling.py` : analyse glissante des episodes de stress.
- `generate_rich_stress_report.py` : generation d'un rapport enrichi de stress test.

## Role de ce bloc

Ces scripts permettent d'expliquer comment la replication se comporte lorsque le regime de marche devient plus instable.
