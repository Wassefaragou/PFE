# Dossier 02 - Optimisation et robustesse

Ce dossier regroupe les scripts qui prolongent ou verifient le moteur principal.

## Fichiers

- `masi20_advanced_te.py` : optimisation avancee du tracking error.
- `masi20_extreme_te.py` : recherche exhaustive sur les combinaisons de titres.
- `study_rebalancing_impact.py` : etude de la sensibilite a la frequence de rebalancing.
- `gen_turnover_exhaustif.py` : analyse du turnover des portefeuilles.
- `merge_results.py` : fusion de fichiers de resultats.
- `audit_v2.py` : audit technique du script principal v2.
- `plot_best_advanced_te.py` : visualisation des meilleures configurations.

## Role de ce bloc

Ces scripts servent a :

- ameliorer la qualite de replication ;
- tester la robustesse des choix ;
- comparer plusieurs variantes ;
- controler la coherence des resultats.
